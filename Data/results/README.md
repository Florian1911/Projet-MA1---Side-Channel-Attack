# Results

Ce dossier contient les petits resultats versionnes qui documentent les essais du rapport:

- `figures/` : figures PNG/SVG conservees dans Git.
- `json/` : resumes, rangs, analyses et petits datasets JSON.
- `campaigns/` : sorties organisees par campagne experimentale.
- `report_results/` : elements directement utilises pour la section resultats.
- `raw/` : petites traces texte/CSV conservees comme reference.

Les traces lourdes (`.npz`, `.npy`, captures completes) restent locales et sont ignorees par Git.
