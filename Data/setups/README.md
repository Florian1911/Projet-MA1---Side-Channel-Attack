# Setups

Setups experimentaux complets et documentes:

- `base_saine_pico5000/` : base minimale PicoScope 5000A pour repartir proprement sur AES no-UART.
- `aes_uart_highside_chb_trigger/` : montage high-side avec UART et trigger sur canal PicoScope.

Chaque sous-dossier contient son propre README avec cablage, firmware et commandes de capture.
